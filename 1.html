TAG

  • 진하게 : <strong></strong>
    Ex)
    strong -> strong
  • 밑줄(under line) : <u></u>
    Ex)
    under line -> under line
  • 제목 : <h1></h1> ~ <h6></h6>
    Ex)

    <h1>Title

    <h2>Title

    <h3>Title

    <h4>Title

    <h5>Title
    <h6>Title
  • 줄넘김 : <br>
    Ex)
    ABCD
    EFGH
  • 목차(list) : <li></li>
  • 목차 그룹(Ordered List) : <ul></ul>
    Ex)
    • 가나다
    • 라마바
    • 사아자
    • 차카타
  • 목차 숫자그룹(Unordered List) : <ol></ol>
    Ex)
    1. 가나다
    2. 라마바
    3. 사아자
    4. 차카타
  • 단락 : <p></p>
  • 링크 : <a></a>
    Ex) (페이지 이동)NAVER
    Ex) (새탭)NAVER
    Ex) (툴팁)NAVER
  • 페이지 제목 : <title></title>
  • 페이지 저장 형식 지정 : <meta charset="utf-8">

문법

  • 속성(Attribute) : 태그만으로 정보가 부족할때 부가적으로 의미를 부여하는 것
    Ex) <img width = "100%" src="html이미지.svg">
